# Задачи для Codex — SNAF STUDIO

## Контекст проекта
Одностраничный лендинг веб-дизайн студии SNAF STUDIO. Статический сайт без сборщика: `index.html`, модульный CSS в `styles/`, JS в `js/main.js`. Язык — русский. Домен — `snafstudio.ru` (куплен, сайт ещё не опубликован). Автор — Иван Смертин, самозанятый.

## Задачи

### 1. Сгенерировать OG-обложку (`images/og-cover.png`)
Создать статичное изображение 1200×630px для превью в соцсетях. Стилистика: фон — градиент `#0ea5e9` → `#2dd4bf`, текст белый, Montserrat 800. Содержание: логотип `</>` слева, текст «SNAF STUDIO — дизайн и создание сайтов» по центру. Формат PNG, вес <300KB. Файл сохранить в `images/og-cover.png`. Мета-теги `og:image` и `twitter:image` в `index.html` уже ссылаются на этот путь.

### 2. Создать полноценный набор favicon
Заменить текущую inline SVG-заглушку в `index.html` на полноценный набор иконок. Дизайн: скруглённый квадрат с акцентным фоном `#0ea5e9`, символ `</>` белым. Сгенерировать:
- `favicon.ico` (16×16, 32×32)
- `favicon-32x32.png`
- `favicon-16x16.png`
- `apple-touch-icon.png` (180×180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest`

Все файлы положить в корень проекта. Обновить `<head>` в `index.html`: убрать текущий `<link rel="icon" href="data:image/svg+xml,...">` и подключить новые файлы.

### 3. Создать `privacy.html` — Политика конфиденциальности
Создать страницу `privacy.html` в корне проекта. Содержание — стандартная политика конфиденциальности для самозанятого (РФ), который собирает имя/телефон/email через формы обратной связи и мессенджеры. Реквизиты: Иван Смертин, самозанятый, email: smertin.lingv@gmail.com. Стилизовать страницу в тех же стилях что и основной сайт (`styles/main.css`), добавить шапку с логотипом и ссылку «Назад на главную». Язык — русский.

### 4. Создать `offer.html` — Договор-оферта
Создать страницу `offer.html` в корне проекта. Содержание — публичная оферта на оказание услуг по дизайну и вёрстке сайтов. Исполнитель: Иван Смертин, самозанятый. Стоимость: 2 000 ₽ за один блок любой сложности. Порядок работы: бриф → дизайн → вёрстка → запуск. Стилизовать аналогично `privacy.html`. Язык — русский.

### 5. Создать `robots.txt` и `sitemap.xml`
**robots.txt** в корне проекта:
```
User-agent: *
Allow: /
Disallow: /legacy/
Disallow: /Documentation/
Sitemap: https://snafstudio.ru/sitemap.xml
```

**sitemap.xml** в корне проекта. Включить:
- `https://snafstudio.ru/` (главная, приоритет 1.0)
- `https://snafstudio.ru/privacy.html` (приоритет 0.3)
- `https://snafstudio.ru/offer.html` (приоритет 0.3)

### 6. Создать страницу 404
Создать `404.html` в корне проекта. Минималистичная страница в стиле сайта: логотип, заголовок «Страница не найдена», подпись «Возможно, она была перемещена или удалена», кнопка «На главную» (ссылка на `/`). Подключить `styles/main.css` и Google Fonts. Без JS.

### 7. Cookie-баннер
Добавить cookie-баннер в `index.html` (внизу страницы, поверх контента). Текст: «Мы используем файлы cookie для улучшения работы сайта. Продолжая использовать сайт, вы соглашаетесь с [политикой конфиденциальности](privacy.html).» Кнопка «Принять». При нажатии: сохранять флаг в `localStorage`, скрывать баннер, не показывать повторно. Стилизация:
- Позиция: `fixed`, `bottom: 0`, `z-index: 200`
- Фон: `var(--surface-overlay-96)`, `backdrop-filter: blur(12px)`
- Скругление верхних углов: 20px
- Кнопка: `.btn--primary` стиль

CSS добавить в отдельный файл `styles/cookie.css`, подключить через `@import` в `main.css` перед `responsive.css`. JS добавить в `js/main.js` внутри `DOMContentLoaded`.

### 8. Подготовить подключение Яндекс.Метрики
Добавить в `index.html` перед закрывающим `</body>` комментарий-плейсхолдер:
```html
<!-- Яндекс.Метрика: вставить код счётчика после получения ID -->
```
В `js/main.js` подготовить функцию `initAnalytics()` (вызывается в конце `DOMContentLoaded`), которая:
- Проверяет наличие `ym` (Яндекс.Метрика) в `window`
- Если есть — отправляет цель `'cta_click'` при клике на любую `.btn--primary`
- Если нет — молча пропускает

Не вставлять сам счётчик (ID ещё нет), только подготовить инфраструктуру.
